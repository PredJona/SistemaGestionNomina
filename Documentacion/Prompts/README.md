# Prompts de trabajo para NomiCore

Esta carpeta contiene prompts reutilizables para continuar el desarrollo, hacer revisiones y preparar la sustentacion de **ProyFinal_LPI_Eq01_NomiCore**.

Los prompts estan escritos para pedir cambios concretos, conservar Windows Forms Designer y proteger la logica de negocio existente. Antes de aplicar un cambio, revise siempre el resultado de `git status` y no incluya credenciales reales en una conversacion.

| Archivo | Uso principal |
|---|---|
| `01_Revision_Tecnica.md` | Revisar errores, riesgos y pruebas faltantes. |
| `02_Mejoras_WinForms.md` | Mejorar formularios sin romper el Designer. |
| `03_Pruebas_y_Validaciones.md` | Crear o ejecutar pruebas seguras. |
| `04_Documentacion_y_UML.md` | Mantener UML, resultados y documentacion. |
| `05_Sustentacion.md` | Preparar respuestas para la exposicion. |
| `06_Git_y_Entrega.md` | Organizar commits y entrega final. |
| `07_Logica_De_Nomina.md` | Cambiar reglas de negocio sin romper nomina, permisos ni trazabilidad. |
