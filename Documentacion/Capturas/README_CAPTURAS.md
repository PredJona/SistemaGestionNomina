# Capturas Reales Para La Entrega

Esta carpeta queda preparada para evidencias reales. No se deben usar capturas falsas, imagenes generadas ni capturas de otro sistema.

## Orden Requerido

1. `01_Login.png`: pantalla de inicio de sesion.
2. `02_Dashboard.png`: pantalla principal o dashboard despues de ingresar.
3. `03_AcercaDe.png`: formulario Acerca de con datos del curso y equipo.
4. `04_Empleados.png`: modulo de registro o listado de empleados.
5. `05_Asistencia.png`: modulo de asistencia con registros visibles.
6. `06_Nomina.png`: calculo de nomina con detalle y totales.
7. `07_Comprobantes.png`: comprobantes generados.
8. `08_PrintPreview.png`: vista previa de impresion con PrintDocument.
9. `09_Reportes.png`: modulo de reportes.
10. `10_Exportacion.png`: mensaje o archivo generado de Excel/PDF.
11. `11_Salir.png`: confirmacion del boton Salir.

## Instrucciones

- Ejecutar la aplicacion desde Visual Studio 2022 o desde el ejecutable compilado.
- Iniciar sesion con `admin/admin123`.
- Navegar por cada modulo y tomar la captura cuando se vea claramente la funcionalidad.
- Guardar cada imagen con el nombre exacto indicado.
- Insertar las capturas en `Proy2_Eq__Resultados.docx` antes de entregar.
